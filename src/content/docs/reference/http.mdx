## Overview

The HTTP service simulates a basic web server. It has two modes of operation:

<Tabs>
<TabItem label="Fake">
In "Fake" mode, requests to any URL will return a generic response with a configurable status code, headers and body. 
</TabItem>
<TabItem label="Real">
In "Real" mode, a directory can be specified and the server will serve static files from that directory. If a requested file is not found, a 404 response will be returned.

<Aside type="caution">"Real" mode is still in development. Please see the [GitHub Issue](https://github.com/lachlanharrisdev/gonetsim/issues/13) for its current status of development</Aside>
</TabItem>
</Tabs>


## Configuration

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `enabled` | bool | `true` | Whether the HTTP service is enabled |
| `listen` | string | `:8080` | The address and port to listen on (e.g. `:8080`) |
| `status` | int | `200` | The status code to return in "Fake" mode |

To see instructions on how to configure these options, see the [configuration guide](/guides/configuration).
