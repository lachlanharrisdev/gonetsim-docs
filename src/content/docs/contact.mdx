Most of  your inquiries will be answered in [this documentation](/). If you believe information is missing from these docs, you are more than welcom to contribute by clicking the "Edit this page" button at the bottom of every page. For bug reports, feature requests, and other project-related discussions, please use the [GitHub repository](https://github.com/lachlanharrisdev/gonetsim).

More direct, serious or private inquiries should be directed to <a href="mailto:gonetsim[at]lachlanharris[dot]dev">gonetsim[at]lachlanharris[dot]dev</a>. This includes feedback, partnerships, legal inquiries, enterprise support & more.

For security vulnerabilities, please use the [GitHub Security Advisory](https://github.com/lachlanharrisdev/gonetsim/security/advisories) to report the issue privately. Do not report security vulnerabilities through public channels. 
If the vulnerability is urgent, or you are unable to use the Github security advisory system, please email us with the address above.
